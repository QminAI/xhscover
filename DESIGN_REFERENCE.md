# XHS Cover Generator - 设计参考分析

## 整体布局
- **页面结构**：单页应用，从上到下分为多个功能区块
- **颜色方案**：红色主题（#FF0000 或接近的红色）、白色背景、黑色文字
- **头部**：品牌名称 + 登录按钮 + 积分显示
- **主要内容区**：三个步骤的流程式设计

## 三步流程设计
1. **上传素材** - 人像/主体（必填）+ 空镜/背景（可选）
2. **选择风格** - 预设风格网格展示 + 我的风格库
3. **详细配置** - 标题、副标题、字体、比例、装饰、其他要求

## 关键设计特点
- **红色虚线边框**：用于强调输入区域和重要内容
- **步骤编号**：红色圆形标签标记每个步骤（1、2、3）
- **风格预览**：3列网格布局展示预设风格
- **表单设计**：清晰的标签、输入框、下拉菜单
- **CTA 按钮**：红色主按钮"生成封面"，位于页面底部
- **提示信息**：灰色文字说明和提示

## 交互特点
- 预设风格可点击放大查看
- 支持多选风格
- 批量模式开关
- 字体风格预览
- 图片比例选择器

## 响应式设计
- 左侧为表单输入区
- 右侧为预览或结果区
- 移动端可能为上下堆叠

## 色彩系统
- 主色：红色（#FF0000 或 #E63946）
- 背景：白色或浅灰色
- 文字：黑色或深灰色
- 强调：红色虚线、红色标签
- 边框：红色虚线（2px dashed）
